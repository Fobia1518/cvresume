# Personal Resume Website

This project is my personal resume/portfolio website.

It is based on the **Start Bootstrap Resume** template, but has been **fully customized and adapted** to reflect my own professional profile, design preferences, and content.

## Base Template

This project is built on top of the following template:

👉 https://startbootstrap.com/template-overviews/resume/

Created by Start Bootstrap.

From this base, I implemented several changes, including:

- Complete content customization (experience, skills, projects)
- Design and color adjustments
- Removal and restructuring of original sections
- Code and style optimizations
- Adaptation to my own development workflow
